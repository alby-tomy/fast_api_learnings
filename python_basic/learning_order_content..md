# Python Basics to Lists

## Follow the learning Order

printing\
comments\
variables\
data types\
input\
type conversion\
operators\
strings\
conditions\
loops\
functions\
lists

------------------------------------------------------------------------

## 1. Printing

``` python
print("Hello, World!")
```

------------------------------------------------------------------------

## 2. Comments

``` python
# This is a comment
print("Comments help explain code")
```

------------------------------------------------------------------------

## 3. Variables

``` python
name = "Alby"
age = 25
print(name, age)
```

------------------------------------------------------------------------

## 4. Data Types

``` python
x = 10          # int
y = 3.14        # float
z = "Hello"     # string
a = True        # boolean
```

------------------------------------------------------------------------

## 5. Input

``` python
name = input("Enter your name: ")
print("Hello", name)
```

------------------------------------------------------------------------

## 6. Type Conversion

``` python
num = int("10")
print(num + 5)
```

------------------------------------------------------------------------

## 7. Operators

``` python
a = 10
b = 5

print(a + b)
print(a - b)
print(a * b)
print(a / b)
```

------------------------------------------------------------------------

## 8. Strings

``` python
text = "Python"
print(text.upper())
print(text.lower())
print(text[0])
```

------------------------------------------------------------------------

## 9. Conditions

``` python
age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

------------------------------------------------------------------------

## 10. Loops

``` python
for i in range(5):
    print(i)

count = 0
while count < 3:
    print(count)
    count += 1
```

------------------------------------------------------------------------

## 11. Functions

``` python
def greet(name):
    return "Hello " + name

print(greet("Alby"))
```

------------------------------------------------------------------------

## 12. Lists

``` python
numbers = [1, 2, 3, 4]

print(numbers[0])
numbers.append(5)
print(numbers)
```
